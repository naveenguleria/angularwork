Angular - TABLE OF CONTENTS

  • ANGULAR COMMANDS
    1 Software needed NODEJS, ANGULAR-CLI, RXJS, VSCODE (EDITOR), GIT (Version Control)
    2 Install NODEJS Download from nodejs.org and install
    3 Install ANGULARJS (globally) npm install -g @angular/cli
    4 Create ANGULARJS app ng new my-app
    ng new my-app --style=scss --routing
    (--routing is added because we want a project that has different page URL's)
    5 Development SERVER cd my-app
    ng serve | ng serve --open | ng serve -o
    6 Create MODULE ng generate module module-name
    ng g m module-name
    7 Create COMPONENT ng generate component component-name
    ng g c component-name
    8 Create SERVICE ng generate service component-name/service-name
    ng g s component-name/service-name
    9 BUILD ng build (Development build)
    ng build --prod
    10 RUN ng serve | ng serve --open | ng serve -o
    11 RUN using local http-server npm install -g http-server
    RUN HTTP SERVER:: http-server dist/build-proj

  • INTERPOLATION { { } }
    1 Interpolation Binding { { nameVar } }
    2 Method Interpolation { { method() } }
    3 Cannot do assignment { { a = 3+3 } } will not work!!!
    4 We cannot access global variables { { window.location.href } } will not work!!!
  • PROPERTY BINDING - Bind value using angular to make ui dynamic
    1 Interpolation Binding - { } Works only for strings (boolean will not work) <input disabled="{ { isDisabled } }"> will not work!!!
    2 Property Binding [ ] <input [disabled]="isDisabled">
    <input [id]="myId">
    3 Two-Way Property Binding [( )] <input [(ngModel)]="txtVal-variable">
    4 bind-attributeName <input bind-disabled="isDisabled">
  • CLASS BINDING - Bind CSS classes using angular to make ui dynamic
    1 [class] <input [class]="classVar" >
    classVar = property defined as typescript variable (*.component.ts)
    1 [class.className] <input [class.txt-red]="isError" >
    isError = property defined as boolean typescript variable (*.component.ts)
    1 [ngClass] - Apply multiple classes conditionally or normally <input [ngClass]="myClasses" >
    myClasses = property defined as boolean typescript variable (*.component.ts)
    myClasses = { "txt-success": this.success, "special": this.special }
  • STYLE BINDING - Bind inline CSS style using angular to make ui dynamic
    1 [style.Property]="'colorName'" - Direct <h4 [style.color]="'red'"> My color is RED </h4>
    1 [style.Property]="myColor" - Using variable <h4 [style.color]="myColor"> My color is RED </h4>
    myColor="orange"; (*.component.ts)
    2 [style.Property]="conditionalExpression" - Using expression <h4 [style.color]="hasError ? 'red' : 'green'"> My color is RED </h4>
    hasError = true; (*.component.ts)
    2 [ngStyle]="styleObjectVar" - Using ngStyle <h4 [ngStyle]="titleStule"> My color is RED </h4>
    titleStule = { color: "blue" ; fontStyle: "italic" } (*.component.ts)
  • EVENT BINDING - Bind events using angular to make ui dynamic
    1 (click) <input type="button" (click)="clickMe()" >
    clickMe() { console.log("I m clicked") }
    1 (click) with $event to send event information <input type="button" (click)="clickMe($event)" >
    clickMe(event) { console.log("I m clicked") }
    1 (click) inline expression <input type="button" (click)="myVar='Welcome Naveen'" >
  • TEMPLATE REFERENCE VARIABLES - #varName
    1 #varName <input type="text" #myName >
    <input type="button" (click)="logMe(myName.value)" >
    logMe(val){ console.log(val); }
    2 Output
  • TWO-WAY data binding - [(ngModel)]
